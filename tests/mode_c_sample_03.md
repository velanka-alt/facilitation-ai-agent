# Mode C Sample 03

## Input
- Problem(문제/상황): 팀 내 갈등으로 협업 속도가 느려짐
- Outcome(원하는 변화/목표): 관계 회복과 협업 규칙 재합의
- Audience(대상/인원/직급/부서): 실무자 12명
- Time(총 시간/기간): 2시간
- Format(운영 형태: 오프라인/온라인/하이브리드): 온라인
- Budget(예산): 없음
- Past Experience(과거 유사 워크숍 경험): 분기 회고 워크숍 경험 있음
- Sensitive Issues(민감 이슈): 성과 책임 배분
- Priority(우선순위: 합의/속도/창의/관계 등): 관계 회복/심리적 안전
- Constraints/Risks(제약/리스크): 발언 침묵, 감정 과열

## Expected Output
1) 추천 옵션 (A/B/C)
- 옵션 A:
  - 목적: 감정/사실/해석을 분리해 갈등을 정리하고 규칙을 재합의한다.
  - 핵심 도구/기법: MDFP, Cluster, Fist to five
  - 예상 산출물: 핵심 이슈 요약, 협업 규칙 합의안
- 옵션 B:
  - 목적: 참가자 흐름에 따라 갈등을 탐색하고 통찰을 도출한다.
  - 핵심 도구/기법: Dynamic Facilitation, Sensing
  - 예상 산출물: 우려/해결 방향 보드, 통찰 리스트
- 옵션 C:
  - 목적: 협업 리듬과 피드백 규칙을 빠르게 정비한다.
  - 핵심 도구/기법: 3F, List, Multi Voting
  - 예상 산출물: 리듬/피드백 규칙, 우선순위 항목

2) 추천 근거
- 근거 1: 감정 이슈가 커서 구조화된 회고 도구가 필요하다.
- 근거 2: 심리적 안전을 확보한 뒤 규칙 합의로 연결해야 한다.
- 근거 3: 2시간 제한이라 핵심 단계만 짧게 구성해야 한다.

3) Draft 아젠다(안)
- 00:00~00:10: 오프닝/안전 규칙 합의
- 00:10~00:40: MDFP 공유(사실-감정-해석-제안)
- 00:40~01:05: 공통 이슈 묶기 (Cluster)
- 01:05~01:30: 협업 규칙 재정의 (List)
- 01:30~01:50: 합의 확인 (Fist to five)
- 01:50~02:00: 클로징/다음 액션

4) 추가 확인 질문 (5개)
- Q1: 갈등이 가장 자주 발생하는 업무 상황은 무엇인가요?
- Q2: 성과 책임에서 가장 민감한 쟁점은 무엇인가요?
- Q3: 기존 회고 워크숍에서 효과적이었던 방식은 무엇인가요?
- Q4: 온라인 참여에서 불편하거나 제한되는 점이 있나요?
- Q5: 합의 결과를 실행 점검할 주기는 어떻게 정하나요?

5) 다음 단계 안내
- CTA: 위 정보를 주시면 2시간 맞춤 아젠다와 진행 문안을 제공합니다.
