# Mode C Sample 03

## Input
- Problem: 팀 내 갈등으로 협업 속도가 느려짐
- Outcome: 관계 회복과 협업 규칙 재합의
- Audience: 실무자 12명
- Time: 2시간
- Budget: 없음
- Past Experience: 분기 회고 워크숍 경험 있음
- Sensitive Issues: 성과 책임 배분

## Expected Output
1) 추천 옵션
- 옵션 1: 관계 회복 대화 + 협업 규칙 재정의 워크숍
- 옵션 2: 갈등 원인 분리 및 합의 프레임 워크숍
- 옵션 3: 협업 리듬/피드백 규칙 재설정 워크숍

2) 추천 근거
- 근거 요약: 갈등과 책임 이슈가 핵심이므로 안전한 대화 구조와 규칙 재정의가 우선.

3) Draft 아젠다
- 하이라이트: 안전 규칙 합의(15분) -> 갈등 요인 분리(40분) -> 협업 규칙 재정의(40분) -> 다음 액션(25분)

4) 추가 확인 질문 (5개)
- Q1: 갈등이 주로 발생하는 상황은 언제인가요?
- Q2: 성과 책임 이슈에서 가장 민감한 쟁점은 무엇인가요?
- Q3: 기존 회고 워크숍에서 효과적이었던 방식은 무엇인가요?
- Q4: 외부 이해관계자와 연결된 이슈가 있나요?
- Q5: 합의 결과를 실행할 책임자는 누구인가요?

5) 다음 단계 안내
- CTA: 위 정보를 주시면 2시간 맞춤 아젠다와 진행 문안을 제공합니다.
