# Mode A Sample 03

## Input
- 목적: 부서 간 갈등 이슈를 정리하고 의사결정 원칙 합의
- 참석자 수: 15명
- 총 시간: 3시간
- 회의 상황/배경: 예산 배분을 두고 이해관계 충돌
- 언어: 한국어
- 운영 형태(온라인/오프라인/하이브리드): 오프라인
- 참석자 구성/레벨(직군/직급/경력 혼합 여부): 부서장/팀장 혼합
- 원하는 결과물 유형(아젠다/질문/스크립트/캔버스): 아젠다/스크립트
- 진행 난이도(초/중/고): 고
- 분위기(방어적/열린/갈등): 갈등
- 제약/주의사항(시간/민감 이슈/의사결정권 등): 예산 배분 이슈 민감, 의사결정권자 모두 참여 필요

## Expected Output
B) 세션 설계팩
1) 시간표 아젠다
- 00:00~00:15: 오프닝/규칙 합의/목표 재확인
- 00:15~01:00: 관점 정리(부서별 주장과 근거)
- 01:00~01:40: 공통 목적/비타협 영역 분리
- 01:40~02:30: 의사결정 원칙 합의
- 02:30~03:00: 합의문 정리 및 다음 액션

2) 활동별 목적/방법/준비물/멘트
- 활동 1: 목적 감정 고조 방지 / 방법 규칙 합의 후 발언 순서 고정 / 준비물 규칙 카드 / 멘트 "지금은 사실과 근거만 말하고, 평가는 잠시 미룹니다."
- 활동 2: 목적 공통 목적 도출 / 방법 부서별 주장 분리 작성 후 교차 확인 / 준비물 보드, 포스트잇 / 멘트 "상대 부서의 말을 먼저 재진술하고, 확인을 받겠습니다."

3) 리스크 포인트 및 대응
- 리스크: 감정 격화로 대화 중단 가능
- 대응: 규칙 위반 시 즉시 중단 후 휴식, 발언 순서 재정렬
