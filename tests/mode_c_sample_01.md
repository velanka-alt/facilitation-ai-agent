# Mode C Sample 01

## Input
- Problem: 부서 간 협업이 느리고 책임이 불명확함
- Outcome: 협업 프로세스와 R&R 합의
- Audience: 매니저 6명, 실무자 12명
- Time: 3시간
- Budget: 없음
- Past Experience: 유사 워크숍 경험 없음
- Sensitive Issues: 성과 책임 배분

## Expected Output
1) 추천 옵션
- 옵션 1: 협업 프로세스 맵핑 + 병목 제거 워크숍
- 옵션 2: R&R 합의 중심 정렬 워크숍
- 옵션 3: 협업 규칙/의사결정 원칙 수립 워크숍

2) 추천 근거
- 근거 요약: 문제의 핵심이 책임 불명확과 흐름 지연이므로, 프로세스 시각화와 R&R 합의가 우선.

3) Draft 아젠다
- 하이라이트: 프로세스 맵핑(60분) -> 병목/책임 도출(45분) -> 합의 규칙 작성(45분) -> 다음 액션(30분)

4) 추가 확인 질문 (5개)
- Q1: 현재 가장 느린 협업 단계는 어디인가요?
- Q2: 최종 의사결정자는 누구로 합의되어 있나요?
- Q3: 부서 간 공유해야 할 핵심 정보는 무엇인가요?
- Q4: 성과 책임 이슈에서 민감한 포인트는 무엇인가요?
- Q5: 워크숍 결과를 승인할 주체는 누구인가요?

5) 다음 단계 안내
- CTA: 위 질문에 답을 주시면 3시간 아젠다를 상세화해 드리겠습니다.
