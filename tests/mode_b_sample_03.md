# Mode B Sample 03

## Input
- 지금 상황: 문제정의 중 특정 팀의 책임 문제가 거론되어 분위기가 예민함
- 현재 단계: 문제정의
- 제한 조건: 민감 이슈, 실명 언급 금지 요청

## Expected Output
1) 개입 멘트
- 멘트 1: 실명 없이, 사실 기반으로만 이야기하겠습니다. 과정과 구조에 집중하겠습니다.
- 멘트 2: 관찰된 사실만 적어주세요.
- 멘트 3: 문제를 해결 가능한 문장으로 바꿔 정리하겠습니다.

2) 다음 질문
- Q1: 특정 주체 없이도 동일하게 발생하는 현상은 무엇인가요?
- Q2: 문제를 측정 가능한 문장으로 바꾸면 어떻게 표현할 수 있나요?
- Q3: 우리가 통제할 수 있는 요인은 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 사실/영향만 분리 기록 -> 원인 분류
- B: 익명 입력으로 문제 문장 수집
- C: 해결 가능 영역과 통제 불가 영역 분리

4) 리스크 및 대응
- 리스크: 책임 공방으로 논의가 중단될 수 있음
- 대응: 규칙 재확인 후 사실 기반으로 재정의 진행
