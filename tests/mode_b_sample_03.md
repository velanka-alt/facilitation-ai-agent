# Mode B Sample 03

## Input
- 지금 상황: 오프닝에서 분위기가 방어적이고 참여 의지가 낮음
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 오프닝
- 제한 조건: 오프닝 10분 내 마무리

## Expected Output
1) 개입 멘트
- 멘트 1: "오늘은 평가가 아니라 개선을 위한 시간입니다. 안전하게 의견을 내실 수 있도록 하겠습니다."
- 멘트 2: "맞고 틀린 답은 없고, 현장의 경험이 가장 중요합니다."
- 멘트 3: "간단히 기대 한 가지씩만 나누고 바로 본론으로 가겠습니다."

2) 다음 질문
- Q1: 오늘 꼭 얻고 싶은 결과 한 가지는 무엇인가요?
- Q2: 이 주제에서 가장 조심스러운 부분은 무엇인가요?
- Q3: 오늘 논의에서 지켜야 할 규칙이 있다면 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 기대 한 줄 공유 -> 규칙 2개 합의
- B: 익명 포스트잇으로 우려 수집 -> 규칙 합의
- C: 체크인 생략 -> 목적/결과물만 명확히 공유

4) 리스크 및 대응
- 리스크: 방어적 분위기로 발언이 나오지 않음
- 대응: 익명 수집 또는 짧은 체크인으로 심리적 안전 확보
