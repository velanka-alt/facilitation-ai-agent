# Mode A Sample 01

## Input
- 목적: 신규 프로젝트 킥오프에서 팀의 목표와 역할을 정렬
- 참석자 수: 10명
- 총 시간: 2시간
- 회의 상황/배경: 부서 혼합, 이전에 협업 경험 적음
- 언어: 한국어
- 운영 형태(온라인/오프라인/하이브리드): 오프라인
- 참석자 구성/레벨(직군/직급/경력 혼합 여부): 팀장 2명, 실무자 8명
- 원하는 결과물 유형(아젠다/질문/스크립트/캔버스): 아젠다/질문/캔버스
- 진행 난이도(초/중/고): 중
- 분위기(방어적/열린/갈등): 열린
- 제약/주의사항(시간/민감 이슈/의사결정권 등): 역할 합의는 초안 수준까지

## Expected Output
A) 가이드 패키지
1) Step-by-step 진행 흐름
- 단계 1: 오프닝 및 목적/규칙 합의
- 단계 2: 현재 기대/우려 수집 (개인 작성 -> 공유)
- 단계 3: 공동 목표/역할 키워드 도출
- 단계 4: 합의사항 정리 및 다음 액션 확정

2) 질문 리스트
- Q1: 이번 프로젝트에서 반드시 성공으로 보고 싶은 한 가지는?
- Q2: 역할/책임이 불명확했던 경험이 있다면 무엇이었나?
- Q3: 협업을 매끄럽게 만드는 규칙 한 가지는?

3) 캔버스/템플릿
- 항목 A: 프로젝트 목표 (성공 정의)
- 항목 B: 역할/책임 (R&R)
- 항목 C: 협업 규칙/리듬
- 항목 D: 리스크/의존성

4) 다음 선택지
- 옵션 A: 역할-책임 매핑 상세화 세션
- 옵션 B: 리스크/의존성 점검 워크숍
- 옵션 C: 2주 후 회고 미팅 설계

B) 세션 설계팩
1) 시간표 아젠다
- 00:00~00:10: 오프닝/목적 공유/규칙 합의
- 00:10~00:35: 기대/우려 개인 작성 및 공유
- 00:35~01:10: 공동 목표 및 역할 키워드 도출
- 01:10~01:40: R&R 초안 정리
- 01:40~02:00: 합의사항/다음 액션 정리

2) 활동별 목적/방법/준비물/멘트
- 활동 1: 목적 목표/역할 정렬 / 방법 개인 작성 후 라운드 공유 / 준비물 포스트잇, 펜 / 멘트 "먼저 각자 적고 짧게 공유하겠습니다."
- 활동 2: 목적 R&R 합의 / 방법 그룹 토의 후 키워드 클러스터링 / 준비물 보드, 스티커 / 멘트 "중복 키워드는 묶고 핵심만 남깁니다."

3) 리스크 포인트 및 대응
- 리스크: 역할 합의가 추상적 수준에서 끝날 수 있음
- 대응: 역할을 문장으로 정의하고 책임 범위를 체크리스트로 확인
