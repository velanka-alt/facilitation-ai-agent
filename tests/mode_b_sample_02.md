# Mode B Sample 02

## Input
- 지금 상황: 합의 단계에서 찬반이 팽팽하고 대립이 길어짐
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 합의
- 제한 조건: 30분 안에 결론 필요

## Expected Output
1) 개입 멘트
- 멘트 1: "두 의견 모두 프로젝트 성공을 위한 관점입니다. 공통 목표부터 확인하겠습니다."
- 멘트 2: "지금은 쟁점을 분리해서 확인하고, 그 다음 선택 기준을 정하겠습니다."
- 멘트 3: "합의가 어려우면 조건부 합의로 전환할 수도 있습니다."

2) 다음 질문
- Q1: 두 안이 각각 만족시키는 핵심 목표는 무엇인가요?
- Q2: 가장 큰 우려 1가지는 무엇인가요?
- Q3: 조건을 붙인다면 어떤 조건이 필요할까요?

3) 운영 옵션 (Plan A/B/C)
- A: 공통 목표 확인 -> 기준 합의 -> 재투표
- B: 우려 사항만 분리 정리 -> 조건부 합의
- C: 결정권자에게 2안 비교표로 상신

4) 리스크 및 대응
- 리스크: 감정 대립이 심화되어 합의 실패
- 대응: 쟁점 분리 후 시간 박스 설정, 조건부 합의로 전환
