# Mode B Sample 02

## Input
- 지금 상황: 시작이 15분 지연되어 참석자들이 산만함
- 현재 단계: 오프닝
- 제한 조건: 전체 1시간 중 40분만 남음

## Expected Output
1) 개입 멘트
- 멘트 1: 시간이 제한되어 있어 핵심만 진행하겠습니다. 오늘의 목적을 1문장으로 정리하겠습니다.
- 멘트 2: 각자 가장 중요한 기대 한 가지를 적고 바로 공유해주세요.
- 멘트 3: 일정상 토론은 최소화하고 합의만 집중하겠습니다.

2) 다음 질문
- Q1: 오늘 반드시 합의해야 하는 것은 무엇인가요?
- Q2: 시간상 생략 가능한 항목은 무엇인가요?
- Q3: 지금 공유할 핵심 기대 한 가지는?

3) 운영 옵션 (Plan A/B/C)
- A: 3분 무음 작성 -> 라운드 로빈 공유
- B: 목적/결과만 합의하고 후속 미팅 확정
- C: 안건 1개만 선택해 집중 진행

4) 리스크 및 대응
- 리스크: 급하게 진행되어 합의가 표면적일 수 있음
- 대응: 후속 회의 일정과 보완 질문을 즉시 확정
