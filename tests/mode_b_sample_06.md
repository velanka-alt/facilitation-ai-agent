# Mode B Sample 06

## Input
- 지금 상황: 회고 단계에서 "시간 낭비"라는 냉소 발언 발생
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 마무리
- 제한 조건: 종료 15분 전

## Expected Output
1) 개입 멘트
- 멘트 1: "그렇게 느끼실 수 있습니다. 무엇이 달라지면 의미 있다고 느끼실까요?"
- 멘트 2: "남은 시간은 15분입니다. 바로 실행 가능한 것 하나만 정하겠습니다."
- 멘트 3: "오늘 나온 내용 중 꼭 지켜야 할 한 가지를 합의하겠습니다."

2) 다음 질문
- Q1: 오늘 논의에서 즉시 실행 가능한 항목은 무엇인가요?
- Q2: 의미 있다고 느끼려면 어떤 변화가 필요할까요?
- Q3: 실행을 책임질 담당자는 누구인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 액션 1개만 확정 -> 책임자/기한 지정
- B: 우려사항 1개 기록 -> 후속 점검 일정 합의
- C: 냉소 발언의 이유를 짧게 수집 -> 다음 회의 의제로 이관

4) 리스크 및 대응
- 리스크: 마무리 불신 확산
- 대응: 실행 가능한 1개에 집중하고 책임자/기한 명확화
