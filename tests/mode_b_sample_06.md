# Mode B Sample 06

## Input
- 지금 상황: 임원 참석으로 발언이 위축되고 상향 의견이 나오지 않음
- 현재 단계: 아이데이션
- 제한 조건: 권력관계 강함, 익명 의견 요청

## Expected Output
1) 개입 멘트
- 멘트 1: 지금은 직급과 무관하게 아이디어를 수집하는 시간입니다.
- 멘트 2: 먼저 익명으로 의견을 모으고, 그다음에 묶어서 공유하겠습니다.
- 멘트 3: 오늘은 양을 확보하고, 평가는 다음 단계에서만 하겠습니다.

2) 다음 질문
- Q1: 지금 개선할 수 있는 작은 시도 한 가지를 적어주세요.
- Q2: 현재 방식의 가장 큰 비효율/불편은 무엇인가요?
- Q3: 안전하게 시도할 수 있는 대안은 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 익명 입력 -> 클러스터링 -> 키워드만 공유
- B: 직급 혼합 2인 짝토의 -> 짝당 1개 발표
- C: 임원 발언 순서 마지막 고정

4) 리스크 및 대응
- 리스크: 동조 편향으로 아이디어 다양성이 낮아질 수 있음
- 대응: 익명 수집과 발언 순서 조정으로 심리적 안전 확보
