# Mode B Sample 04

## Input
- 지금 상황: 두 가지 안으로 의견이 갈려 결론이 나지 않음
- 현재 단계: 합의
- 제한 조건: 의사결정까지 15분만 남음

## Expected Output
1) 개입 멘트
- 멘트 1: 선택 기준을 2개로 좁혀 빠르게 비교하겠습니다.
- 멘트 2: 기준에 따라 점수를 매기고 합산하겠습니다.
- 멘트 3: 오늘은 결정 범위를 줄여 합의하고, 최종 결정자는 합의된 기준에 따라 결정합니다.

2) 다음 질문
- Q1: 두 안을 가르는 핵심 기준은 무엇인가요?
- Q2: 각 안의 리스크는 무엇인가요?
- Q3: 오늘 결정해야 하는 범위는 어디까지인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 기준 2개 점수화 -> 합산
- B: 3점 투표로 우선순위 결정
- C: 오늘 결정 범위를 축소하고 후속 검토 일정 확정

4) 리스크 및 대응
- 리스크: 시간 부족으로 합의가 강제될 수 있음
- 대응: 결정 범위를 축소하거나 후속 검토를 명시
