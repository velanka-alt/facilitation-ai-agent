# Mode B Sample 04

## Input
- 지금 상황: 논의가 산으로 가고 핵심 주제를 놓침
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 문제정의
- 제한 조건: 남은 시간 25분

## Expected Output
1) 개입 멘트
- 멘트 1: "잠깐 멈추겠습니다. 지금 논의 중인 주제를 다시 확인하겠습니다."
- 멘트 2: "현재 주제와 다른 논점은 파킹랏에 적고, 핵심으로 돌아가겠습니다."
- 멘트 3: "남은 시간은 25분이므로 핵심 질문에 집중하겠습니다."

2) 다음 질문
- Q1: 오늘 반드시 답해야 하는 질문은 무엇인가요?
- Q2: 지금 나온 이슈 중 핵심과 관련된 것은 무엇인가요?
- Q3: 파킹랏으로 넘길 이슈는 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 파킹랏 정리 -> 핵심 질문 1개만 깊게 논의
- B: 핵심 질문 2개만 선택 -> 각 10분 타임박스
- C: 핵심 정의만 합의 -> 다음 회의로 이관

4) 리스크 및 대응
- 리스크: 핵심 합의 없이 종료
- 대응: 질문 1개로 축소하고 타임박스 운영
