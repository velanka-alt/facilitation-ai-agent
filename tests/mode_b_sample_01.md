# Mode B Sample 01

## Input
- 지금 상황: 아이데이션 단계에서 2명만 계속 말하고 나머지는 침묵
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 아이데이션
- 제한 조건: 20분 남음, 대표자 참석(직급 차이 존재)

## Expected Output
1) 개입 멘트
- 멘트 1: "지금은 아이디어를 최대한 넓게 모으는 단계입니다. 먼저 조용히 3분간 적고 공유하겠습니다."
- 멘트 2: "앞에서 나온 내용과 다른 아이디어가 한 가지만 더 나와도 좋습니다."
- 멘트 3: "대표자분들은 마지막에 정리 의견을 주시면 됩니다. 먼저 현장 의견부터 들어볼게요."

2) 다음 질문
- Q1: 지금 문제를 해결하기 위한 아이디어를 3개만 적어주세요.
- Q2: 방금 적은 아이디어 중 실행 난이도가 낮은 것은 무엇인가요?
- Q3: 기존과 다른 관점에서 떠오르는 대안이 있나요?

3) 운영 옵션 (Plan A/B/C)
- A: 3분 개인 작성 -> 라운드 로빈 공유
- B: 2인 페어 토크 -> 페어 대표 1개만 공유
- C: 포스트잇 벽면 부착 -> 전원 묵음 읽기

4) 리스크 및 대응
- 리스크: 직급 차이로 발언이 다시 편중될 수 있음
- 대응: 발언 순서를 명시하고, 대표자는 마지막 순서로 배치
