# Mode A Sample 02

## Input
- 목적: 분기 목표를 공유하고 우선순위를 정리
- 참석자 수: 8명
- 총 시간: 90분
- 회의 상황/배경: 최근 프로젝트 실패 경험으로 방어적 분위기
- 언어: 한국어
- 운영 형태(온라인/오프라인/하이브리드): 온라인
- 참석자 구성/레벨(직군/직급/경력 혼합 여부): 동일 팀, 실무자 중심
- 원하는 결과물 유형(아젠다/질문/스크립트/캔버스): 질문/캔버스
- 진행 난이도(초/중/고): 초
- 분위기(방어적/열린/갈등): 방어적
- 제약/주의사항(시간/민감 이슈/의사결정권 등): 실패 책임 추궁 금지, 익명 의견 허용

## Expected Output
A) 가이드 패키지
1) Step-by-step 진행 흐름
- 단계 1: 안전 규칙 합의(비난 금지, 사실 중심)
- 단계 2: 현재 목표 인식 확인(개인 작성 -> 공유)
- 단계 3: 우선순위 기준 합의
- 단계 4: 상위 목표 3개 확정 및 다음 액션

2) 질문 리스트
- Q1: 이번 분기에 꼭 달성해야 하는 목표 한 가지는?
- Q2: 실패를 반복하지 않으려면 어떤 기준이 필요할까?
- Q3: 오늘 합의해야 할 우선순위 기준은 무엇인가?

3) 캔버스/템플릿
- 항목 A: 목표 후보 리스트
- 항목 B: 우선순위 기준(가치/리스크/기간)
- 항목 C: 상위 목표 3개
- 항목 D: 합의 근거 요약

4) 다음 선택지
- 옵션 A: 목표별 성공 지표 정의 세션
- 옵션 B: 리스크 요인 사전 점검 미팅
- 옵션 C: 2주 단위 실행 계획 수립
