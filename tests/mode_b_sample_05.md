# Mode B Sample 05

## Input
- 지금 상황: 마무리 단계에서 피로도가 높고 일부가 먼저 나가려 함
- 현재 단계: 마무리
- 제한 조건: 남은 시간 10분

## Expected Output
1) 개입 멘트
- 멘트 1: 10분 안에 합의와 다음 액션만 정리하겠습니다.
- 멘트 2: 오늘 워크숍에서 느낀점을 짧게 작성하고 공유하겠습니다.
- 멘트 3: 후속 일정과 책임자를 바로 확정합니다.

2) 다음 질문
- Q1: 오늘 합의된 핵심 결론은 무엇인가요?
- Q2: 다음 액션의 책임자는 누구인가요?
- Q3: 후속 확인이 필요한 사항은 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 결론 1문장 라운드 -> 액션 3개 확정
- B: 결론/액션만 정리하고 회고는 서면으로 대체
- C: 액션과 일정만 확정하고 종료

4) 리스크 및 대응
- 리스크: 액션이 구체화되지 않고 종료될 수 있음
- 대응: 책임자/기한을 반드시 지정
