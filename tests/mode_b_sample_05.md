# Mode B Sample 05

## Input
- 지금 상황: 임원이 중간에 들어와 방향을 바꾸려 함
- 현재 단계(오프닝/문제정의/아이데이션/합의/마무리): 아이데이션
- 제한 조건: 참석자 긴장, 발언 위축

## Expected Output
1) 개입 멘트
- 멘트 1: "오신 김에 지금까지 나온 내용을 1분만 요약드리겠습니다."
- 멘트 2: "먼저 현장 의견을 충분히 듣고, 마지막에 정리 의견을 부탁드려도 될까요?"
- 멘트 3: "지금은 아이디어 수집 단계라 평가보다는 다양한 의견을 모으겠습니다."

2) 다음 질문
- Q1: 지금까지 나온 아이디어 중 추가로 보완할 점이 있나요?
- Q2: 현장에서 가장 실현 가능성이 높은 아이디어는 무엇인가요?
- Q3: 임원 관점에서 고려해야 할 리스크는 무엇인가요?

3) 운영 옵션 (Plan A/B/C)
- A: 5분 휴식 후 임원과 1:1 조율 -> 원래 흐름 유지
- B: 임원 의견을 옵션으로 기록 -> 전체 비교
- C: 소그룹 전환 후 각 그룹 요약 공유

4) 리스크 및 대응
- 리스크: 임원 발언으로 참여자 침묵 심화
- 대응: 발언 순서 조정, 소그룹 전환으로 참여 회복
