# Mode A Sample 05

## Input
- 목적: 원격 근무 팀의 커뮤니케이션 규칙을 정리
- 참석자 수: 7명
- 총 시간: 90분
- 회의 상황/배경: 메시지 응답 지연과 회의 피로가 누적됨
- 언어: 한국어
- 운영 형태(온라인/오프라인/하이브리드): 온라인
- 참석자 구성/레벨(직군/직급/경력 혼합 여부): 팀장 1명, 실무자 6명
- 원하는 결과물 유형(아젠다/질문/스크립트/캔버스): 질문/캔버스
- 진행 난이도(초/중/고): 초
- 분위기(방어적/열린/갈등): 열린
- 제약/주의사항(시간/민감 이슈/의사결정권 등): 발언이 적은 구성원 배려, 익명 의견 허용

## Expected Output
A) 가이드 패키지
1) Step-by-step 진행 흐름
- 단계 1: 오프닝/목적 공유/온라인 진행 규칙 안내
- 단계 2: 불편 사례 수집(온라인 보드 무음 작성 -> 공유)
- 단계 3: 커뮤니케이션 규칙 후보 정리
- 단계 4: 실험 규칙 2~3개 확정 및 점검 일정 합의

2) 질문 리스트
- Q1: 최근 가장 불편했던 커뮤니케이션 순간은?
- Q2: 응답 속도나 채널 사용에서 꼭 지켜야 할 기준은?
- Q3: 이번 주부터 시도할 작은 규칙 한 가지는?

3) 캔버스/템플릿
- 항목 A: 불편 사례(상황/영향)
- 항목 B: 원인/패턴
- 항목 C: 합의할 규칙(채널/응답/회의)
- 항목 D: 2주 실험안

4) 다음 선택지
- 옵션 A: 2주 후 규칙 점검 미팅
- 옵션 B: 채널별 가이드 문서화
- 옵션 C: 회의 피로도 줄이기 세션
