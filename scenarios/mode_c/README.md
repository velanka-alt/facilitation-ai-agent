# Mode C - Workshop Recommender

## Who/When
- 워크숍 의뢰자(리더/HR/프로젝트 오너)
- 우리 팀에 필요한 워크숍이 불명확할 때

## Input
- 문제/상황, 원하는 변화/목표
- 대상(인원/직급/부서), 시간/기간
- 옵션: 운영 형태, 예산, 과거 경험, 민감 이슈, 우선순위, 제약/리스크

## Output
- 추천 옵션 1~3개(목적/도구/산출물 포함)
- 추천 근거 요약
- Draft 아젠다(시간표)
- 추가 확인 질문 5개
- 다음 단계 안내
- 도구/기법 명칭은 `docs/tool_frame_reference.md` 기준
