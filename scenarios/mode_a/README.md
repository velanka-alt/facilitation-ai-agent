# Mode A - Framework Provider

## Who/When
- 초중급 FT, HR/교육 담당자
- 설계를 시작할 때 막히는 순간

## Input
- 필수: 목적, 참석자 수, 총 시간, 회의 상황/배경, 언어
- 옵션: 운영 형태(온라인/오프라인/하이브리드), 참석자 구성/레벨, 결과물 유형, 난이도, 분위기, 제약/주의사항, 복잡 조건 여부(하이브리드/다회차/장시간(8시간 이상)/영어/이해관계 복잡)

## Output
A. 가이드 패키지 (기본)
- Step-by-step 진행 흐름
- 질문 리스트
- 캔버스 템플릿
- 다음 선택지 제안

B. 세션 설계팩 (선택, 아젠다/스크립트 요청 시)
- 시간표 아젠다
- 활동별 목적/방법/준비물/멘트
- 리스크 포인트 및 대응 문구

결과물 유형 매핑
- 질문/캔버스: A
- 아젠다/스크립트: B
- 복수 선택: A+B
- 결과물 유형 미입력: A 기본 제공

## 적용 범위
- 단일 세션 기반의 기본 워크숍/회의 설계에 적합하다.
- 하이브리드, 다회차/장시간(8시간 이상), 영어 진행, 이해관계가 복잡한 설계는 전문 퍼실리테이터 컨설팅을 권장한다.
- 위 조건이 입력되면 결과물에 "전문 설계/컨설팅 권고" 문구를 포함한다.
