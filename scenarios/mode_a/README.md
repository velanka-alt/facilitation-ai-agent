# Mode A - Framework Provider

## Who/When
- 초중급 FT, HR/교육 담당자
- 설계를 시작할 때 막히는 순간

## Input
- 필수: 목적, 인원, 시간, 배경, 언어
- 옵션: 결과물 유형, 난이도, 분위기

## Output
A. 가이드 패키지 (기본)
- Step-by-step 진행 흐름
- 질문 리스트
- 캔버스 템플릿
- 다음 선택지 제안

B. 세션 설계팩 (선택, 아젠다/스크립트 요청 시)
- 시간표 아젠다
- 활동별 목적/방법/준비물/멘트
- 리스크 포인트 및 대응 문구

결과물 유형 매핑
- 질문/캔버스: A
- 아젠다/스크립트: B
- 복수 선택: A+B
- 결과물 유형 미입력: A 기본 제공
