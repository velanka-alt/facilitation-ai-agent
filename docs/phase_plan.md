# Phase Plan

Phase 0: 설계 기준 고정
- 산출물: 미션/가드레일/용어/시나리오 템플릿
- 성공 기준: 비개발자도 README만 보고 흐름 설명 가능

Phase 1: 텍스트 MVP (Mode A)
- 산출물: Step-by-step 흐름, 질문 리스트, 캔버스 템플릿
- 성공 기준: 80% 재사용률

Phase 2: 설계팩 자동화
- 산출물: Detail Agenda, 진행 멘트, 리스크 대응 문구
- 성공 기준: 실제 워크숍 3건 이상 적용

Phase 3: Mode C + Mode B
- 산출물: 추천 리포트, 실시간 개입 멘트/옵션
- 성공 기준: 현장 활용성 검증 완료

Phase 4: 확장 기능
- 산출물: 파일 업로드 분석, 보고서 자동화, 성장 미션, 채널 연동(예: Slack/카카오톡)
- 성공 기준: 유지비용 대비 가치 검증
