퍼실리테이션 AI 에이전트: 워크숍의 모든 순간을 함께하는 코파일럿
워크숍의 기획, 운영, 회고 전 과정을 보조하는 AI 에이전트의 핵심 기능과 운영 원칙을 한눈에 전달합니다.

Mode A: 워크숍 설계 가이드
- Input: 목적, 인원, 시간, 배경, 언어, (옵션) 결과물 유형/난이도/분위기
- Output: Step-by-step 진행 흐름, 질문 리스트, 캔버스, 다음 선택지
- (선택) 세션 설계팩: 시간표 아젠다, 활동별 목적/방법/준비물/멘트, 리스크/대응

Mode B: 실시간 현장 개입
- Input: 상황 요약, 현재 단계(오프닝/문제정의/아이데이션/합의/마무리), 제한 조건
- Output: 개입 멘트 2~3개, 다음 질문 3개, 운영 옵션 A/B/C, 리스크 및 대응

Mode C: 맞춤형 프로그램 추천
- Input: 문제상황, 원하는 변화, 대상(인원/레벨), 시간, (옵션) 예산/과거 경험/민감 이슈
- Output: 추천 옵션 1~3개 + 근거, Draft 아젠다, 추가 확인 질문 5개, 다음 단계(CTA)

운영 가드레일
- 중립성 유지
- 입력 범위 내 답변
- 가정 명시
- 고위험 자문 금지

발전 로드맵 (Phase 0~4)
- Phase 0: 문서 기반 설계
- Phase 1: 텍스트 MVP
- Phase 2: 설계팩 자동화
- Phase 3: Mode B/C 확장
- Phase 4: 파일 분석/보고서 자동화/채널 연동 등 확장
