# Tool/Frame Keywords

이 문서는 툴/프레임 매칭을 위한 키워드 사전이다.
툴 이름은 `tool_frame_reference.md`의 고정 표기를 그대로 사용한다.

## 사용 원칙
- 키워드는 쉼표로 구분한다.
- 유사 개념(한/영/약어)을 함께 넣는다.
- 매칭 시 툴 이름 자체도 키워드로 간주한다.
