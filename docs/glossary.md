# Glossary

- FT: Facilitator, 퍼실리테이터
- CA: Company Facilitator, 사내 퍼실리테이터
- Mode A: Framework Provider (설계 가이드)
- Mode B: Co-Facilitator (실시간 대응)
- Mode C: Workshop Recommender (추천 리포트)
- 센싱: 사전 인터뷰/진단을 통해 맥락을 확보하는 단계
- 가드레일: 안전성과 책임 경계를 정의하는 운영 원칙
- 세션 설계팩: 아젠다와 활동 상세(목적/방법/준비물/멘트) 및 리스크 대응을 포함한 설계 산출물
- CTA: Call to Action, 다음 단계 안내 문구
